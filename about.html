I made this website so I can make games more fun with friends. The first page is a hub for the whole website. There is a link on the first page called 'Roulette', this will take you to another page which will have a display of games with an image of each game underneath its name, if you were to click on any game it will present the title of a strat along with the rules. A strat is a set of rules that the players must follow, strats can make the game more difficult or makes the group of friends work more as a team. Also, there is a 'Custom' link on every page, the page will allow you to type in your own strats for any game and then it chooses at random.
